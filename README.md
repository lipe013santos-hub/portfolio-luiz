# Portfólio Luiz Felipe Santos

Versão 3.0 unificada.

## Estrutura

- `index.html` — Home do portfólio
- `case-01.html` — Case profissional de otimização de custos bancários
- `case-02.html` — Case acadêmico de Business Analytics com IA
- `dashboard-porsche.html` — Dashboard comercial do Case #02
- `style.css` — Único arquivo de estilos do portfólio
- `assets/` — Imagens e evidências dos projetos

## Próximos projetos

Para adicionar um novo projeto, copie a estrutura de `case-02.html`, altere textos/imagens e adicione um novo card em `index.html`.
