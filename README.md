# Portfólio Luiz Felipe Santos

Portfólio profissional com foco em Análise Financeira, Business Analytics, tratamento de dados e IA aplicada à produtividade analítica.

## Estrutura final

- `index.html` — página inicial
- `case-01.html` — projeto profissional de otimização de custos bancários
- `case-02.html` — projeto acadêmico de dashboard comercial com IA
- `dashboard-porsche.html` — dashboard interativo do Case #02
- `style.css` — arquivo único de estilos
- `assets/` — imagens e evidências dos projetos

## Padrão para próximos projetos

Para criar um novo case, copie a estrutura do `case-02.html`, renomeie para `case-03.html` e adicione um novo card na seção de projetos da Home.
