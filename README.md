# Portfólio Luiz Felipe

Site estático em HTML/CSS.

## Como visualizar
Abra o arquivo `index.html` no navegador.

## Como publicar grátis no GitHub Pages
1. Crie uma conta no GitHub.
2. Crie um repositório chamado `portfolio-luiz`.
3. Envie todos os arquivos desta pasta.
4. Vá em Settings > Pages.
5. Em Source, selecione `Deploy from a branch`.
6. Escolha a branch `main` e pasta `/root`.
7. Salve e aguarde o link ser gerado.

Depois, use esse link no LinkedIn, currículo e Google Sites.
